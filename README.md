# 🤖 AI Modern Website

This project is a modern AI-themed website created using HTML, CSS, and JavaScript.
The website provides information about Artificial Intelligence and includes multiple pages for navigation.

## 🌐 Live Website

https://spiffy-kringle-6171ae.netlify.app/

## 📌 Features

* Modern and clean website design
* Responsive layout
* Multiple pages navigation
* AI information and content
* Contact page for users
* Simple and user-friendly interface
